## HR看中的

### 教育经历
所学专业, 科目, 有无数学等基础.

### 工作经历
上份工作内容, 主要解决什么问题, 做的怎么样, 是否出彩.

### 研究方向/经验所在
是否对某一技术有**较深入**的研究.

### 编程语言
涉及到了实践, 一般都是要求静态类型语言, 如 C++, Java 和一门脚本语言, 如 Python, 或比较火的语言, 如 Go, Erlang等.

### 框架使用
本质上来说, 对框架认识是否深入决定了对框架背后理论的理解深度, 实际工作中, 框架必不可少, 置于跳出框架, 那是中高手的事.
对于自己来说, 要有点追求, 对框架的实现要有点了解.

### 一些通用 base 技能
如设计模式, 数据结构, 算法, 计算机网络, 并发等通用知识
**业务逻辑实现能力, 包括设计与实现**

### 思维能力
是否有着清晰的逻辑, 这个完全要靠不断的练习提高, 侧重于解决一件问题的方法论,  将大问题分解为一个个小问题.

### 通用技能
对自身很重要的品质, 沟通能力, 表达能力, 演说能力, 文档写作能力, 团队合作, 学习能力, 语言能力(英语)
