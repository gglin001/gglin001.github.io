进入我的主页
使用Chrome浏览器的开发者工具(F12进入), 在Console一栏输入代码
 ``` js
var btns = document.getElementsByClassName("is-active");
console.log(btns.length)
for(var i=0; i<btns.length; ++i){
    btns[i].click();
//    setTimeout(100);
}
```
回车!

其实就是模拟点击, 要注意的是run一遍不能完全取消当前页面已加载的赞, 可以多运行几遍, 看到输出btns.length值比较小时即可, 然后刷新页面, 再来一遍.
